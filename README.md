Um projeto desenvolvido durante o **DevClub** que utiliza **Inteligência Artificial** para gerar landing pages completas a partir de um prompt.

O usuário descreve um negócio (ex: cafeteria, mercado, academia) e a aplicação envia a descrição para uma IA, que retorna uma página profissional em **HTML + CSS**, pronta para visualização em tempo real.

---

## 💡 Como Funciona

1. Usuário digita um prompt
2. JavaScript captura o texto
3. Requisição é enviada para IA
4. IA gera HTML + CSS
5. Resultado é exibido em código + preview
